# 商标政策

此[商标政策](https://www.percona.com/trademark-policy)旨在确保 Percona 品牌产品或服务的用户知道他们收到的产品确实由 Percona 开发、批准、测试和维护。商标通过区分一家公司或个人的产品和服务与另一家公司的产品和服务，有助于防止市场混淆。

Percona 拥有许多商标，包括但不限于 Percona、XtraDB、Percona XtraDB、XtraBackup、Percona XtraBackup、Percona Server 和 Percona Live，以及与这些商标相关的独特视觉图标和徽标。Percona 的未注册和已注册商标均受保护。

未经 Percona 书面许可，不得在任何产品、服务、网站或其他用途的名称、URL 或其他识别特征中使用任何 Percona 商标，但以下三个有限例外情况除外。

*首先*，在对真实的 Percona 产品进行名义合理使用时，您可以使用相应的 Percona 商标。

*其次*，当 Percona 根据 GNU 通用公共许可证（"GPL"）版本发布产品时，您可以在根据 GPL 的条款和条件分发该产品的逐字副本时使用相应的 Percona 商标。

*第三*，您可以引用已修改的 GPL 发布的 Percona 软件分发版使用相应的 Percona 商标，前提是这些修改仅是为了允许软件在 Percona 尚未发布软件的操作系统或硬件平台上运行而进行的微小更改，并且这些第三方更改不会影响软件的行为、功能、特性、设计或性能。获得此 Percona 品牌软件的用户收到的是 Percona 软件的实质精确实现。

Percona 保留随时自行决定撤销此授权的权利。例如，如果 Percona 认为您的修改超出了本政策中授予的有限许可范围，或者您对 Percona 商标的使用对 Percona 有害，Percona 将撤销此授权。撤销后，您必须立即停止使用适用的 Percona 商标。如果您在撤销后未立即停止使用 Percona 商标，Percona 可能会采取行动保护其在 Percona 商标中的权利和利益。Percona 不授予对 Percona 软件的任何其他修改版本使用任何 Percona 商标的许可；此类使用需要我们的事先书面许可。
