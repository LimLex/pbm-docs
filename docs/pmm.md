# 通过 Percona Monitoring and Management 管理备份

您不仅可以通过命令行管理备份，还可以通过 Web 界面使用 [Percona Monitoring and Management (PMM) :octicons-link-external-16:](https://www.percona.com/doc/percona-monitoring-and-management/2.x/index.html) 进行管理。这样您就不必在多个服务器上手动运行命令。相反，您可以从一个地方安排备份或按需运行备份，还可以接收实时监控警报。

查看 PMM 文档以获取以下指南：

* [如何配置 PMM 以监控 MongoDB :octicons-link-external-16:](https://docs.percona.com/percona-monitoring-and-management/3/install-pmm/install-pmm-client/connect-database/mongodb.html)
* [PMM 中 MongoDB 的备份管理 :octicons-link-external-16:](https://docs.percona.com/percona-monitoring-and-management/3/backup/mongodb-backup/backup_mongo.html)。 

备份遇到问题？查看[通过 Percona Monitoring and Management 故障排除备份管理](troubleshoot/pbm-pmm.md) 以获取帮助。
