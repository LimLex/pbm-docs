# *Percona Backup for MongoDB* 1.6.1 (2021-11-04)

| 发布日期 | 2021 年 11 月 4 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |

Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或单个副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

* 弃用对 MongoDB 3.6 的支持。Percona Backup for MongoDB 仍与 MongoDB 3.6 和 Percona Server for MongoDB 3.6 兼容；但是，不再针对此版本测试进一步的增强和错误修复。

* 通过使用顺序 `delete-pitr`/`install-backup` 操作而不是内存中的备份意图，改进了备份和 PITR 例程的对齐。这修复了备份无法启动的问题。

* 添加了使用 EC2 实例配置文件自动访问 S3 存储桶的支持。当使用 EC2 实例部署 Percona Backup for MongoDB 时，EC2 环境变量和元数据用于 S3 身份验证，使您无需在 Percona Backup for MongoDB 配置文件中显式指定 S3 凭据。这对于使用 Amazon EC2、kiam、kube2iam 或 irsa 等服务部署的架构很有用。

* 扩展了 `pbm-agents` 的日志记录。这改善了 Percona Backup for MongoDB 的用户体验。
