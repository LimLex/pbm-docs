# Percona Backup for MongoDB 2.0.2 (2022-10-27)

| 发布日期 | 2022 年 10 月 27 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

此版本通过为 JSON 格式的状态输出提供错误消息（[PBM-988](https://jira.percona.com/browse/PBM-988)）修复了与 Percona Backup for MongoDB 一起运行的应用程序的可用性问题。它还包括针对安全漏洞问题 [CVE-2022-32149](https://nvd.nist.gov/vuln/detail/CVE-2022-32149) 的修复。

## 已修复的错误

[PBM-988](https://jira.percona.com/browse/PBM-988)：为 `pbm status` 命令的 JSON 输出显示错误消息
