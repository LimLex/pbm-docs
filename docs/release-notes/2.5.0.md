# Percona Backup for MongoDB 2.5.0 (2024-05-16)

[Installation](../installation.md){.md-button}


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份，并将这些备份恢复到特定时间点。

## 发布亮点

* 您现在可以恢复所需的[包含用户和角色的自定义数据库](../features/selective-backup.md#restore-a-database-with-users-and-roles) 子集。这对于每个用户都有单独数据库并针对其进行身份验证的部署很有用。
* 以前版本的 PBM 要求 MongoDB 中的 `readConcern` 和 `writeConcern` 设置为 `majority`。您现在可以显式覆盖此行为，从而确保在配置为在没有多数的情况下运行或由于某种原因丢失多数的集群中进行备份。


## 新功能
