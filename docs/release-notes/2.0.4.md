# Percona Backup for MongoDB 2.0.4 (2023-02-21)

| 发布日期 | 2023 年 2 月 21 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

[指定 `mongod` 二进制文件的自定义路径](../usage/restore-physical.md#define-a-mongod-binary-location) 的能力简化了物理恢复过程。

* [PBM-1047](https://jira.percona.com/browse/PBM-1047) - 修复了新环境中物理恢复失败的问题。
