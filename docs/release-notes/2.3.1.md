# Percona Backup for MongoDB 2.3.1 (2023-12-04)

[Installation](../installation.md){.md-button}


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

* 添加了对 Percona Server for MongoDB 7.0 的支持。
* 使用 Microsoft Azure Blob Storage 进行备份时定义自定义端点的能力
* 改进了 PBM Docker 镜像，允许使用共享的 `mongodb` 数据卷进行物理备份
* 更新了 Golang 库，包括针对安全漏洞 CVE-2023-39325 的修复。


## 改进

* [PBM-1195](https://jira.percona.com/browse/PBM-1195) - 允许 Azure 存储的自定义 endpointUrl
* [PBM-1205](https://jira.percona.com/browse/PBM-1205) - 将 PBM Docker 镜像中的用户更改为 `mongod`


## 已修复的错误

* [PBM-1193](https://jira.percona.com/browse/PBM-1193) - 修复了与 PSMDB 连接丢失时 `pbm-agent` 失败的问题
