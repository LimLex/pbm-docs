# *Percona Backup for MongoDB* 1.2.0

| 发布日期 | 2020 年 5 月 13 日   |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或单个副本集）中创建一致备份的工具。

## 新功能

* [PBM-348](https://jira.percona.com/browse/PBM-348)：添加删除旧备份的能力

* [PBM-447](https://jira.percona.com/browse/PBM-447)：pbm-speed-test：添加一个工具来现场测试压缩和上传速度

## 改进

* [PBM-431](https://jira.percona.com/browse/PBM-431)：通过压缩调整、并行化提高转储输出速度

* [PBM-461](https://jira.percona.com/browse/PBM-461)：s2 被设置为默认压缩机制

* [PBM-429](https://jira.percona.com/browse/PBM-429)：将定期备份进度消息添加到 pbm-agent 日志

* [PBM-140](https://jira.percona.com/browse/PBM-140)：添加了取消备份的能力

## 已修复的错误

* [PBM-451](https://jira.percona.com/browse/PBM-451)：如果存储类型设置为文件系统，重新同步不起作用
