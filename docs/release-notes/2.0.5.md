# Percona Backup for MongoDB 2.0.5 (2023-03-23)

| 发布日期 | 2023 年 3 月 23 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

此版本引入了以下修复：

* 对于 mongod `--dbpath` 选项的最后一个字符为斜杠 ('/') 的部署的物理恢复过程
* 通过将 Golang 库依赖项更新到最新版本来修复安全漏洞 [CVE-2022-41723](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2022-41723)

## 已修复的错误

* [PBM-1058](https://jira.percona.com/browse/PBM-1058) - 修复了如果 `--dbpath` 选项有尾随斜杠的物理恢复问题。

* [PBM-1076](https://jira.percona.com/browse/PBM-1076) - 由于 CVE-2022-41723 更新 golang.org/x/net 模块
