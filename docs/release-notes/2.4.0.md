# Percona Backup for MongoDB 2.4.0 (2024-03-05)

[Installation](../installation.md){.md-button}


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

* 您现在可以[删除特定类型的备份快照](../usage/delete-backup.md#__tabbed_2_3)。例如，仅删除您可能已创建且不再需要的逻辑备份。您还可以使用新的 `--dry-run` 标志检查将删除的确切内容。此改进可帮助您更好地满足组织的备份策略，并改善您清理过期数据的体验。
* 时间点恢复 oplog 切片现在与备份快照并行运行。这确保您可以从需要数小时才能完成的非常大的备份中对任何时间戳进行[时间点恢复](../usage/pitr-tutorial.md)。
* Percona Backup for MongoDB 软件包现在也可用于 ARM64 架构。因此，用户不仅可以在基于 ARM 的工作站上的 Docker 容器中运行 Percona Backup for MongoDB，还可以在这些工作站上安装软件包。ARM64 软件包适用于以下操作系统：

    * Ubuntu 20.04 (Focal Fossa)   
    * Ubuntu 22.04 (Jammy Jellyfish)
    * Red Hat Enterprise Linux 8 及兼容衍生版本
    * Red Hat Enterprise Linux 9 及兼容衍生版本
