# Percona Backup for MongoDB 1.4.1

| 发布日期 | 2021 年 1 月 28 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或单个副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 改进

* [PBM-621](https://jira.percona.com/browse/PBM-621)：在 `pbm status` 输出中显示不完整的备份

* [PBM-619](https://jira.percona.com/browse/PBM-619)：优化 `pbm status` 的存储响应时间

* [PBM-615](https://jira.percona.com/browse/PBM-615)：检查当前集群的备份有效性

* [PBM-608](https://jira.percona.com/browse/PBM-608)：通过添加 GSSAPI 支持为 PBM 启用 Kerberos 身份验证

* [PBM-478](https://jira.percona.com/browse/PBM-478)：防止从不完整的备份恢复

* [PBM-610](https://jira.percona.com/browse/PBM-610)：修复 `pbm status` 命令的 GCS 响应时间

## 已修复的错误

* [PBM-618](https://jira.percona.com/browse/PBM-618)：在处理之前检查备份快照中的完整文件集
