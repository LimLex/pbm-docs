# Percona Backup for MongoDB 1.6.0

| 发布日期 | 2021 年 8 月 16 日 |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或单个副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点：

* 支持 Percona Server for MongoDB 和 MongoDB Community 5.0

* 时间点恢复增强：能够从任何先前的快照恢复和可配置的 oplog 事件范围

* PBM 命令的 JSON 输出，以简化 PBM 与应用程序的接口
