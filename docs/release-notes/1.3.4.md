# Percona Backup for MongoDB 1.3.4

| 发布日期 | 2020 年 11 月 19 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或单个副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 改进

* [PBM-586](https://jira.percona.com/browse/PBM-586)：在恢复期间为 S3 下载器添加请求超时

* [PBM-584](https://jira.percona.com/browse/PBM-584)：在恢复期间忽略分片配置

## 已修复的错误

* [PBM-555](https://jira.percona.com/browse/PBM-555)：通过分块从 s3 存储下载备份，修复恢复期间的"错误解复用归档"错误

* [PBM-460](https://jira.percona.com/browse/PBM-460)：由于冲突的命名空间目标，恢复失败（感谢用户 pedroalb 报告此问题）
