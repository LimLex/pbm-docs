# Percona Backup for MongoDB 2.2.1 (2023-07-27)

| 发布日期 | 2023 年 7 月 27 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

通过此版本，您可以增加备份启动的等待时间，从而消除创建 `$backupCursor` 需要比平时更长时间时 PBM 失败的问题。


## 改进

* [PBM-1154](https://jira.percona.com/browse/PBM-1154) - 添加配置选项以更改备份启动状态的等待超时

## 已修复的错误

* [PBM-1151](https://jira.percona.com/browse/PBM-1151) - 修复了当 Percona Server for MongoDB 无法打开 `$backupCursor` 时 PBM 崩溃的问题

* [PBM-1155](https://jira.percona.com/browse/PBM-1155) - 在增量备份期间失败时重试打开 `$backupCursor`
