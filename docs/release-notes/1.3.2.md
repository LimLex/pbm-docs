# Percona Backup for MongoDB 1.3.2

| 发布日期 | 2020 年 10 月 14 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或单个副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 新功能

* [PBM-426](https://jira.percona.com/browse/PBM-426)：为 S3 存储桶添加 AWS KMS 密钥加密/解密

配置格式

```yaml
storage:
  s3:
    serverSideEncryption:
      sseAlgorithm: "aws:kms"
      kmsKeyID: "........"
```

（感谢用户 pedroalb 报告此问题）

## 改进
