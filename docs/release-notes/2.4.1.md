# Percona Backup for MongoDB 2.4.1 (2024-03-25)

[Installation](../installation.md){.md-button}


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

此版本修复了增量备份失败的问题。这是由于备份元数据文档达到 16MB 的最大大小限制造成的。通过引入处理元数据文档的新方法修复了此问题：它不再包含备份文件列表，该列表现在单独存储在存储上，并由 PBM 按需读取。新的元数据处理方法适用于物理、增量和基于快照的备份。


## 已修复的错误

* [PBM-1252](https://perconadev.atlassian.net/browse/PBM-1252) - 修复了由于元数据文档达到大小限制而导致大型数据集的增量备份失败的问题
