# {{pbm.full_name}} 2.9.1 ({{date.2_9_1}})

{{pbm.full_name}} 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份，并将这些备份恢复到特定时间点。

此版本修复了阻止 PBM 在使用 MongoDB Community 和 Percona Server for MongoDB 的部署中进行物理备份的问题。
