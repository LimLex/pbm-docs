# Percona Backup for MongoDB 2.0.3 (2023-01-11)

| 发布日期 | 2023 年 1 月 11 日  |
|------------- | ---------------|
| 安装 | [安装 Percona Backup for MongoDB](../installation.md) |


Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或非分片副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。

## 发布亮点

* [增量物理备份](../features/incremental-backup.md) 使您能够降低存储成本并促进业务关键数据的数据安全。仅保存差异可加快完成时间，并使增量备份的大小比完整备份小得多。因此，在云设置的情况下，您可以节省存储空间和数据传输。这是[技术预览功能](../reference/glossary.md#technical-preview-feature)，但我们欢迎您的反馈以改进功能。
* 现在您可以选择性地备份和恢复分片集群中的非分片集合。这扩展了要处理的数据集。
* 添加对 AWS IRSA（服务账户的身份角色）凭据的支持，允许在 Pod 中运行的 Percona Backup for MongoDB 使用 IAM 角色访问 AWS 存储。这提高了云基础设施的安全性，并使您能够从单一位置控制对它的访问。
* Percona Backup for MongoDB 现在可在 Red Hat Enterprise Linux 9 及兼容衍生版本上使用
