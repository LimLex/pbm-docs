# Percona Backup for MongoDB 1.0.0

Percona 很高兴于 2019 年 9 月 19 日宣布我们最新软件产品 Percona Backup for MongoDB 1.0.0 的 GA 版本。

Percona Backup for MongoDB 是一个分布式、低影响的解决方案，用于创建 MongoDB 分片集群和副本集的一致备份。这是一个用于在 MongoDB 分片集群（或单个副本集）中创建一致备份并将这些备份恢复到特定时间点的工具。该项目受到（并打算取代）Percona-Lab/mongodb_consistent_backup 工具的启发。

Percona Backup for MongoDB 支持 [Percona Server for MongoDB](https://www.percona.com/software/mongo-database/percona-server-for-mongodb) 或版本 3.6 或更高版本的 [MongoDB Community Server](https://www.mongodb.com/download-center/community)，并启用了 [MongoDB 复制](https://docs.mongodb.com/manual/replication/)。支持的平台的二进制文件以及包含源代码的 tarball 可从 [Percona Backup for MongoDB 下载页面](https://www.percona.com/downloads/percona-backup-mongodb/) 获得。有关 Percona Backup for MongoDB 和安装步骤的更多信息，请参阅文档。

Percona Backup for MongoDB 1.0.0 具有以下特点：

* 与之前的版本相比，Percona Backup for MongoDB 的架构和身份验证已简化。

* 在 [Amazon Simple Storage Service](https://aws.amazon.com/s3/) 或兼容存储（如 [MinIO](https://min.io/)）上存储备份数据。

* **pbm list** 的输出显示从连接的 MongoDB 分片集群或副本集创建的所有备份。
